# SQL Tutorial & Practice Guide

Welcome to your SQL Practice Guide! This folder contains elaborated guides, queries, and exercises based on the core concepts of SQL. 

## 🚀 Best Way to Practice

To truly master SQL, reading isn't enough—you have to write queries. Here is the **best way to practically master everything here**:

### 1. **Online Playgrounds (Fastest Way, No Installation)**
The easiest way to start practicing immediately is using online SQL engines:
- [DB Fiddle](https://www.db-fiddle.com/) - Allows you to define schema and run queries with MySQL, Postgres, or SQLite.
- [SQL Fiddle](http://sqlfiddle.com/) - Another great browser-based SQL environment.
- [LeetCode SQL Practice](https://leetcode.com/problemset/database/) - Fantastic for solving real-world, scenario-based SQL interview questions.

### 2. **Local Installation (For Real Development)**
Set up a database on your PC to get the real developer experience:
- **MySQL / PostgreSQL**: Install MySQL or Postgres locally along with a database GUI like **DBeaver**, **pgAdmin**, or **MySQL Workbench**.
- **SQLite**: The simplest database. You can use **DB Browser for SQLite** without running heavy background services.

### 3. **The Recommended Study Workflow**
1. Read through each module in this folder sequentially (`01_Basics...`, `02_Sorting...`, etc.).
2. Copy the example tables provided in the modules into DB Fiddle.
3. Run the example queries to see the outputs live.
4. Try breaking and modifying the queries (e.g., change the `WHERE` condition, add another grouping, query for different data).
5. Solve the practice challenges at the end of each document!

## 📁 Folder Structure
- `01_Retrieval_and_Filtering.md`: `SELECT`, `WHERE`, `AND/OR`, `BETWEEN`, `IN`, `LIKE`
- `02_Sorting_and_Aggregation.md`: `ORDER BY`, `SUM`, `AVG`, `COUNT`, `GROUP BY`, `HAVING`, `DISTINCT`
- `03_Math_and_Nulls.md`: Arithmetic, Math Functions, `NULL`, `COALESCE`
- `04_Conditional_Logic.md`: `CASE` statements
- `05_Joins.md`: `INNER`, `LEFT`, `RIGHT`, `FULL JOIN`
- `06_Master_Practice.md`: A complete set of exercises simulating real-world scenarios to test your skills!
